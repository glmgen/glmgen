Detailed Description

Utility functions for working with the trendfiltering penalty matrix.

Author
Taylor Arnold, Ryan Tibshirani, Veerun Sadhanala
Date
2014-12-24 The penalty matrix D in the trendfiltering problem has many nice properties which distinquish it from arbitrary sparse of banded matricies. Much of the speed and stability of the trendfiltering algorithms come from taking advantage of these properties. Helper functions for doing so are collected here.
