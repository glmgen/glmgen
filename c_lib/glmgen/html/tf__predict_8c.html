Calculate maximum lambda value. Will return the largest lambda value for which the penalty term is zero for Gaussian losses. Will only be approximate for other loss functions.

Parameters
betathe beta vector for the prediction; length n-k
xthe original positions used in the fit
nnumber of observations
korder of the fit
familyfamily of the fit
x0the new positions to predict at
n0the number of observations in x0
predallocated space for the predicted values
zero_toltolerance for the fitting algorithm; default is 1e-11
Returns
void
Note
The results will not be valid unless the values in x0 are within the range of the original x inputs.
